# ShoexProject

Monorepo for Shoex admin frontend and Django backend.

Folders:
- `shoex-admin/` — React + TypeScript admin dashboard
- `shoex-backend/` — Django backend with GraphQL API

Setup (local):
1. Backend: create and activate a virtualenv, install requirements, run migrations.
2. Frontend: `cd shoex-admin && npm install && npm run dev`.

See project README files in `shoex-admin/README.md` and `shoex-backend/README.md` for more details.
